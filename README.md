# Laravel E-Commerce

A Laravel-based e-commerce application built as a foundation for an online store.  
This project focuses on clean structure, core e-commerce logic, and scalability for future features.

It is designed as a learning + portfolio project, not a finished SaaS product.

---

## Features

- User authentication (register & login)
- Product management (CRUD)
- Shopping cart functionality
- Laravel MVC architecture
- Blade templating
- MySQL database support
- Ready for extension (orders, payments, admin panel)

---

## Tech Stack

- **Backend:** Laravel (PHP)
- **Frontend:** Blade, CSS, JavaScript
- **Database:** MySQL
- **Build tools:** Composer, Vite
- **Version control:** Git & GitHub
